# Stream은 "데이터의 흐름"을 다루는 도구이다.

컬렉션(List, Set 등)에 들어 있는 데이터를 **하나씩 흘려보내면서 처리**하기 위한 개념입니다. (데이터 가공 방식)


```java
products.stream()          // 1. 소스
        .map(ViewProductDTO::new)//2.중간 연산 =.map(product->new ViewProductDTO(product)
        .toList();          // 3. 최종 연산

```

### 소스

소스에서는 아직 아무일도 안일어남

### 중간 연산

중간 연산에는 .map  .filter .sorted를 사용 가능하다.

map - 형태를 바꾼다

```java
.map(Product → ViewProductDTO)
```

filter - 조건으로 거른다

```java
.filter(product -> product.getPrice() > 10000)
```

### 최종 연산

최종 연산에는 toList, forEach, findFirst, count 가 존재한다.

toList - 리스트로 만든다

forEach - 하나씩 처리 (보통 로그, 테스트 출력 등에 사용)

findFirst - 첫번째 요소 가져오기(orElseThrow 같이 쓰기)

count - 개수 확인(Long)
