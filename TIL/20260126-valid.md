# 유효성 검사

```java
1. Bean Validation (javax.validation / jakarta.validation)

@Valid, @NotNull, @Size, @Max, @Min 등의 어노테이션을 DTO나 엔티티 필드에 붙여 조건을 정의합니다.

스프링이 컨트롤러 호출 시 자동으로 검증하고, 조건이 맞지 않으면 예외를 던져줍니다.

import jakarta.validation.constraints.Size;

public class BoardDTO {

    private String title;

    private String description;

    @Size(max = 5, message = "이미지는 최대 5개까지 가능합니다")
    private List<String> images;
}

컨트롤러에서 사용한 예시:

@PostMapping("/addBoard")
public ResponseEntity<?> addBoard(@Valid @RequestBody BoardDTO boardDTO) {
    boardService.addBoard(boardDTO);
    return ResponseEntity.ok().build();
}

이렇게 하면 if (size > 5) 같은 검사를 코드에 직접 쓰지 않아도 된다!
```
